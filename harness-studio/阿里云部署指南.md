# Harness Studio — 阿里云 + Cloudflare 部署指南

## 目标
在阿里云服务器上部署 Harness Studio，通过 Cloudflare 提供免费 HTTPS 域名。

---

## 步骤 1：阿里云服务器配置

### 1.1 创建云服务器
1. 登录阿里云控制台
2. 购买 **ECS 云服务器**（推荐配置：2核4G，CentOS 8）
3. 记录服务器 IP 地址

### 1.2 配置安全组
在 ECS 实例的安全组中添加以下规则：

| 协议 | 端口 | 来源 | 说明 |
|------|------|------|------|
| TCP | 22 | 0.0.0.0/0 | SSH 登录 |
| TCP | 3000 | 0.0.0.0/0 | 应用服务 |

### 1.3 登录服务器
```bash
ssh root@你的服务器IP
```

---

## 步骤 2：安装 Docker

```bash
# 安装依赖
yum install -y yum-utils device-mapper-persistent-data lvm2

# 添加 Docker 源
yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

# 安装 Docker
yum install -y docker-ce docker-ce-cli containerd.io

# 启动 Docker 并设为开机自启
systemctl start docker
systemctl enable docker

# 验证安装
docker --version
```

---

## 步骤 3：部署应用

### 3.1 下载部署脚本
```bash
curl -sSL https://raw.githubusercontent.com/qiufengsir/Harness_Studio/main/harness-studio/deploy.sh -o deploy.sh
chmod +x deploy.sh
```

### 3.2 设置环境变量并运行
```bash
export DEEPSEEK_API_KEY="你的 DeepSeek API Key"
./deploy.sh
```

### 3.3 验证部署
```bash
# 查看日志
docker logs -f harness-studio

# 测试访问
curl http://localhost:3000
```

---

## 步骤 4：Cloudflare 免费域名配置

### 4.1 添加域名到 Cloudflare
1. 打开 https://dash.cloudflare.com
2. 点击 **Add Site**
3. 输入你的域名（如果没有域名，可以在阿里云注册一个）

### 4.2 配置 DNS
在 Cloudflare 的 DNS 页面添加：

| 类型 | 名称 | 内容 | TTL |
|------|------|------|-----|
| A | @ | 你的服务器 IP | Auto |
| A | www | 你的服务器 IP | Auto |

### 4.3 配置反向代理
1. 点击 **Rules** → **Origin Rules**
2. 创建规则：
   - Rule name: `Harness Studio Proxy`
   - When incoming requests match: `Hostname equals your-domain.com`
   - Then: `Set origin` → 输入 `http://你的服务器IP:3000`

### 4.4 开启 HTTPS
1. 点击 **SSL/TLS** → **Overview**
2. 设置为 **Full** 模式
3. 开启 **Always Use HTTPS**

---

## 步骤 5：完成

你的应用现在可以通过以下地址访问：
- HTTP: http://你的服务器IP:3000
- HTTPS: https://你的域名.com

## 常用命令

```bash
# 查看应用日志
docker logs -f harness-studio

# 重启应用
docker restart harness-studio

# 更新应用（重新拉取代码并部署）
export DEEPSEEK_API_KEY="你的密钥"
cd harness-studio && git pull && ./deploy.sh

# 停止应用
docker stop harness-studio

# 删除应用
docker rm harness-studio
```

## 数据备份

```bash
# 备份数据库
docker cp harness-studio:/data/dev.db ./dev.db.bak

# 恢复数据库
docker cp ./dev.db.bak harness-studio:/data/dev.db
docker restart harness-studio
```
